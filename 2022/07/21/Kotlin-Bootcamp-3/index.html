Kotlin-Bootcamp(3)

下面是 學習Kotlin的classes怎麼使用

Kotlin是真的based on java，但是很多東西，有被簡化過後，比如說會預設public，會自動內涵attributes的getter/setter(看起來就像是直接call properties一樣)，但假如說就是想要自己定義getter/setter的話

可見度：private < protected < internal < public(default everywhere)

constructor

Kotlin 可以default多個constructors就是有傳參數的情況(利用init)

假如說想要讓一個class可以被inherit那就需要在class 的前面加上open 關鍵字

kotlin 有分abstract class (有constructor)，和interface

甚至在某些function 的參數傳入的時候，可以規定說傳入的物件必須要先符合另一個interface

如果希望讓class 在inherit 的時候可以有default value，把class 改成object

Kotlin 有專門用來放資料的class，叫做data class
