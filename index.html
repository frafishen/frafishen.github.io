Website Learning Material

這幾天的工作就是先學習。使用網站為MDN Web Docs中的Learn web development。下面列了些我覺得必須要看的一些文章。

Command Line：

在做網頁時，需要測試一些東西，或是引用一些package(別人使用，我們引用的)，這些都需要用到cmd(windows+r，輸入cmd可開啟)，操控cmd的指令就叫做command line

cmd 的速成小教學

Package Management：

寫網頁的時候可以比較容易管理使用的package，檢查更新或是路徑重複等。同時也可以自定命令項及及時自動更新本地伺服器的網頁(不需要一直切來切去按F5)

npm 教學 (一種常見的package manager)
